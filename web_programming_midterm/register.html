Q2: How to create a registration form using HTML and CSS, please explain in your own words.

1101670  Nasa Takigami/ 瀧上凪早

Basically when you create a registration form using HTML and CSS, you need to use <input>.

<input> elements are able to make a user input several information depends on what type of <input> you use.
If you use <input type= " text", the user can type anything to the input space. To tell the user what kind of inforomation to type in, <label > is also used. It usually used together with <input>, because it tells the user what to type into the space nearby.
Therefore, <input type= " text" is used when you want the user to write an open-ended question which cannot be answered with a simple'yes' or 'no', or cannot be selected from the options that we give to the user.
In this assignment, it is user's first and last name and username.

Other than <input type= " text", we have several types that we can use in different ways to get user's information, such as 'radio ', ' checkbox', ' date', ' email', and so on.
I used some of them for making the registration form this time. One of them is < radio> to let the user to choose their gender from 3 options, male, female, and other.
Likewise, I used <checkbox> to check whether the user agreed to the agreement and policy or not, I've been seeing this when I create an accout on any web page, so I added one to mine as well.

Additionally, there is a type " date", I used this to get the user's birthday, when you click on little calender icon on the right, a calender pops up so that you can choose the date from the calender.

Also, type"password" can be used to let the user to type the password, because what they type won&apo;t be shown on the screen, and type"email" as well, it only can be used for emails because it tells you to include '@' in your answer.

When we want give the user options to select one(or more) from, we can use <select> to give options. I used <select> to let the user to choose the country where they from, so I gave 9 Asian country and other for options, and I also wrote the code selected in the value 'Japan', so that the answer is automatically selected Japan even before the user select one.

Last but not least, you have to add a button to submit the registration, mine is linked to another html file which tells the user that they have complete the registration on the previous page.

I added some design to this page, such as background and Bootstrap grid system, so that the design came up with me will be shown the way I want.
